# EXIF 数据可视化工具

一个纯前端的 JPG 图片 EXIF 信息批量读取与可视化分析工具。

## 功能特性

- **批量读取** - 支持同时处理多张 JPG/JPEG 图片
- **文件夹支持** - 可选择整个文件夹，自动扫描子目录
- **拖拽上传** - 支持拖拽文件或文件夹到页面
- **EXIF 解析** - 提取光圈、快门、焦距、ISO、相机型号、拍摄时间等信息
- **数据可视化** - 柱状图展示各参数分布情况
- **统计摘要** - 显示平均光圈、最常用快门、平均焦距等统计数据
- **CSV 导出** - 一键导出所有数据为 CSV 文件
- **完全离线** - 无需网络，纯前端实现

## 使用方法

1. 双击 `index.html` 在浏览器中打开
2. 选择图片或文件夹，或直接拖拽到页面
3. 等待 EXIF 数据读取完成
4. 查看统计图表和详细数据表
5. 可点击"导出 CSV"保存数据

## 技术实现

| 模块 | 技术 |
|------|------|
| EXIF 解析 | 自定义二进制解析器，直接读取 ArrayBuffer |
| 图表绘制 | 纯 Canvas API，无外部依赖 |
| 文件处理 | File API + webkitGetAsEntry |
| 数据导出 | Blob + CSV |

## 性能优化

- 只读取文件头部 64KB，无需加载整张图片
- 并行处理所有文件
- 使用 TextDecoder 快速解码字符串
- 自适应 DPR 高清渲染

- Firefox
